# Portofoliu Lidia Dobos

Proiect de portofoliu construit cu React și Vite. Site-ul prezintă proiecte web, concepte de produs, abilități de design și detalii de contact.

## Comenzi utile

- `npm run dev` pornește serverul local de dezvoltare.
- `npm run build` generează versiunea pentru publicare.
- `npm run lint` verifică fișierele cu regulile ESLint.

## Structură

- `src/App.jsx` conține conținutul principal al portofoliului.
- `src/App.css` și `src/index.css` conțin stilurile.
- `public/` conține imaginile și iconița site-ului.
